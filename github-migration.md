# GitHub Migration Discovery

Prompts for GitHub migration planning and discovery activities.

## Prompts

### Repository Analysis

```
Analyze the repository for migration planning and provide a comprehensive assessment.

OUTPUT REQUIREMENTS:
-   **Repository Overview:** Summary of the repository including size, languages, and structure.
-   **Dependencies:** A list of external dependencies and their versions.
-   **CI/CD Pipelines:** Description of existing automation and workflows.
-   **Migration Risks:** Potential challenges and risks to consider.
-   **Recommendations:** Actionable steps for a successful migration.

Please provide a thorough analysis to support migration planning.
```
