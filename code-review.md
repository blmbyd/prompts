# Code Review

Prompts for code review assistance and feedback generation.

## Prompts

### Code Quality Analysis

```
Review the following code and provide a comprehensive quality analysis.

OUTPUT REQUIREMENTS:
-   **Code Summary:** A brief description of what the code does.
-   **Quality Assessment:** Rate the overall code quality (Excellent/Good/Fair/Poor) with justification.
-   **Issues Found:** A bulleted list of any bugs, anti-patterns, or code smells.
-   **Improvements:** Specific suggestions for improving readability, maintainability, and performance.
-   **Best Practices:** Recommendations for aligning with industry best practices.

Please be constructive and provide actionable feedback.
```
