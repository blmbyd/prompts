# Transcripts

Prompts for general transcript processing and analysis.

## Prompts

### Transcript Analysis

```
Process and analyze the provided transcript to extract key information.

OUTPUT REQUIREMENTS:
-   **Summary:** A brief overview of the transcript content.
-   **Speakers:** List of identified speakers with their roles if mentioned.
-   **Key Points:** A bulleted list of the main topics and points discussed.
-   **Quotes:** Notable or important quotes from the transcript.
-   **Sentiment:** Overall tone and sentiment of the conversation.

Please ensure the analysis is comprehensive and well-organized.
```
