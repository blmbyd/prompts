# Prompts Notebook

| Folder | Description |
|--------|-------------|
| [business-meetings](./business-meetings/) | Prompts for business meeting transcripts and notes |
| [technical-meetings](./technical-meetings/) | Prompts for technical meeting analysis |
| [transcripts](./transcripts/) | Prompts for general transcript processing |
| [github-migration](./github-migration/) | Prompts for GitHub migration discovery |
| [code-review](./code-review/) | Prompts for code review assistance |
| [documentation](./documentation/) | Prompts for documentation generation |
| [emails](./emails/) | Prompts for email drafting and responses |
