# Technical Meetings

Prompts for technical meeting analysis, architecture discussions, and technical decision documentation.

## Prompts

### Technical Discussion Summary

```
Summarize the technical meeting into a structured and actionable report.

OUTPUT REQUIREMENTS:
-   **Meeting Header:** Include Title, Date, Time, and Attendees.
-   **Technical Context:** Brief description of the technical topic or problem discussed.
-   **Key Decisions:** A bulleted list of all technical decisions made.
-   **Architecture Notes:** Any architectural changes or considerations discussed.
-   **Action Items:** A table with columns for Task, Owner, and Deadline for each technical task.
-   **Open Questions:** Unresolved technical questions requiring follow-up.

Please ensure the summary captures all technical details accurately.
```
